# How to use QuizForge with Canvas New Quizzes (7 quick steps)

Screenshots: place the provided PNGs in `User_Docs/screenshots/` with the filenames shown below so they render in this guide.

1) Start a new quiz in Canvas  
Click `+ Quiz` and choose **New Quizzes** when prompted.  
![New quiz button](screenshots/step1_plus_quiz.png)  
![Choose New Quizzes](screenshots/step1_choose_engine.png)

2) Build your quiz in QuizForge  
Paste or upload using the web app, then click **Go**. When it finishes, download the ZIP.

3) Find the downloaded ZIP  
This ZIP contains the Canvas QTI package and print-ready docs.  
![Downloaded ZIP](screenshots/step3_zip_download.png)

4) Extract the ZIP  
Right-click the ZIP and choose **Extract All** so Canvas can read the QTI file cleanly.  
![Extract all](screenshots/step4_extract_all.png)

5) In Canvas, open Import Content  
Inside New Quizzes, open the menu (three dots) and click **Import Content**.  
![Import menu](screenshots/step5_import_menu.png)

6) Import the QTI file  
In the modal, drag/drop or browse to the QTI ZIP from the extracted folder, then click **Import**.  
![Import modal](screenshots/step6_import_modal.png)

7) Open and publish  
Click **Build** to view/edit items, then **Save** or **Save & Publish** when you’re ready.  
![Build button](screenshots/step7_build_button.png)

Tip: If you need to regenerate, just paste a new spec into QuizForge and repeat steps 3–7. Canvas will import it as a new quiz.***
