How It Works

Step 1

Load modules into your AI and talk to it like your personal Teaching Assistant.

Step 2

When you're happy, tell your AI-TA to output the "QuizForge-ready TXT".

Step 3

Put the TXT to QuizForge, click "Queue Quiz", and get your .ZIP package.

Step 4

Import to a New Quiz in Canvas and print your copies! Ready to go!

What’s Inside the ZIP?

  • Canvas Export (QTI)

    Ready-to-import package for New Quizzes. Auto-generated with points and balanced answers.

  • Printable Packet + Key

    DOCX or PDF quiz version, answer key, and rationale companion for quick duplication.

  • Processing Logs

    PASS/WEAK PASS logs detailing auto-fixes and fairness warnings. Fail logs ready to copy into your AI.

  • Answer Distribution

    Optional analytics showing correct answer positions and T/F balance for transparency.

Power Users & Offline Workflow

Prefer to run everything locally? The CLI mirrors the web pipeline.

  1. Clone the repo and set up the Python environment (see README).
  2. Drop TXT files into `DropZone/`, run `python engine/orchestrator.py`.
  3. Collect packages from `Finished_Exports/`—same “UNZIP THIS FIRST” structure.

Need to keep data offline for testing scenarios? We’ll keep this workflow documented alongside the hosted option.
