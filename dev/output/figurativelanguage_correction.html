Q1: <p><strong>Section 1 - Single Application</strong></p><p>The baby blanket felt as soft as a cloud.</p>
Choice✓ / ✗Rationale
ASimile✓This is correct because the sentence compares the blanket to a cloud using the word as, which signals a simile.
BMetaphor✗This is not correct because a metaphor makes a direct comparison without using like or as.
CHyperbole✗This is not correct because the sentence does not exaggerate for effect.
DPersonification✗This is not correct because the blanket is not given human actions or feelings.
